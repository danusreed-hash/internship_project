# Multi-Modal AI Assistant

## Overview

This project is an intelligent Multi-Modal AI Assistant capable of processing and understanding both text and image inputs to provide meaningful responses.

## Key Features

* Upload and analyze images
* Generate descriptive image captions
* Answer questions based on image and text content
* Maintain conversational context with memory
* Provide evidence-based and context-aware responses

## Technologies Used

* Python
* Streamlit
* Hugging Face Transformers
* BLIP (Bootstrapping Language-Image Pre-training) Model
* Pillow (PIL)

## Running the Application

Install the required dependencies:

```bash
pip install -r requirements.txt
```

Launch the application:

```bash
streamlit run app.py
```
